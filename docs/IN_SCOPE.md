# Сайт-визитка — что входит в тариф

## Включено (IN SCOPE)

- До 5 страниц (состав страниц определяется брифом)
- До 7 секций на страницу (базовые: Hero, О себе, Услуги, Контакты, Footer; дополнительные — по брифу)
- Логотип: текстовое (имя/название) или изображение-файл (PNG/SVG) — по брифу
- Мобильная адаптация (от 320px)
- Базовые анимации появления секций (fade/slide при скролле, без библиотек)
- Форма обратной связи (поля: имя + телефон/email + сообщение) → письмо на email клиента — **опционально, по запросу в брифе**
- До 2 шрифтов, до 3 цветов бренда
- Деплой на Vercel / подключение к домену клиента (если DNS настроен самим клиентом)
- Meta-теги: title, description, og:title, og:description, og:image
- **3 итерации правок** — каждая итерация это один список замечаний, переданных за раз

## Не включено (OUT OF SCOPE — оплачивается отдельно)

| Что | Почему отдельно |
|-----|----------------|
| Более 3 страниц | Увеличивает объём разработки |
| Более 7 секций на страницу | Увеличивает объём разработки |
| Сложные анимации (Framer Motion, parallax, Lottie) | +1–2 дня работы |
| Hover-эффекты сложнее стандартных Tailwind transition | Требует дополнительной разработки |
| Мультиязычность | Требует отдельной архитектуры |
| Блог / новости | Требует CMS или бэкенда |
| Авторизация / личный кабинет | Требует бэкенда |
| Интеграция с CRM / мессенджерами | Требует отдельной разработки |
| Разработка логотипа и фирменного стиля | Задача дизайнера |
| SEO-продвижение | Отдельная услуга (включена только базовая техническая оптимизация) |
| Написание текстов и контента | Клиент предоставляет самостоятельно |
| 4-я и последующие итерации правок | Больше 3 раундов |

## Важно

- Клиент предоставляет все материалы (тексты, фото, логотип) **до начала работ**
- Задержка материалов = задержка дедлайна на то же количество дней
- Итерация правок = один список замечаний, переданный **за один раз** (не по одному замечанию в день)
