# Агент: Анализ скоупа

> Используй этот промпт когда клиент предоставил ТЗ или дополнительные пожелания сверх стандартной формы.
> Вставь промпт в Claude вместе с содержимым ТЗ и IN_SCOPE.md конкретного шаблона.

---

## Промпт

```
Ты помогаешь оценить заявку клиента на разработку сайта.

ВХОДНЫЕ ДАННЫЕ:

1. Стандартный скоуп тарифа (из IN_SCOPE.md):
[ВСТАВИТЬ СОДЕРЖИМОЕ docs/IN_SCOPE.md нужного шаблона]

2. ТЗ / дополнительные пожелания клиента:
[ВСТАВИТЬ ТЕКСТ ТЗ ИЛИ ПОЖЕЛАНИЙ]

ЗАДАЧА:

Проанализируй все пункты из ТЗ клиента и раздели их на три группы:

**1. Входит в стандартный тариф**
Перечисли что из ТЗ уже покрывается тарифом без доплаты.

**2. Выходит за рамки тарифа — нужна доплата**
Составь таблицу:

| Пункт из ТЗ | Почему вне тарифа | Оценка (+часов) |
|-------------|------------------|-----------------|
| ...         | ...              | ...             |

Ориентировочные оценки (используй как базу, ±50% в зависимости от деталей):
- Дополнительная страница: +4–8 ч
- Сложные анимации (Framer Motion, parallax): +4–8 ч
- Фильтрация/теги: +8–12 ч
- Dark/Light mode toggle: +4–6 ч
- Многошаговая форма: +6–10 ч
- CMS (клиент сам добавляет контент): +20–40 ч
- Интеграция с платёжной системой: +16–32 ч
- Мультиязычность: +16–24 ч
- Автоматическая email-рассылка: +8–16 ч
- Аналитика (GA4, Pixel) + настройка событий: +4–8 ч
- Блог: +20–40 ч

**3. Требует уточнения**
Что в ТЗ сформулировано неоднозначно — задай клиенту конкретные вопросы.

**Итог**
- Стандартный тариф: ✓
- Доп. разработка: ~X часов → ориентировочная доплата: [оставить пустым — заполнить по своей ставке]
- Вопросы клиенту: [если есть]

Отвечай чётко, по делу. Не перечисляй то, чего нет в ТЗ.
```

---

## Как использовать результат

1. Если out-of-scope пунктов нет → стандартный тариф, переходим к следующему шагу
2. Если есть → обсудить с клиентом:
   - Убрать пункты из скоупа
   - Или согласовать доплату и зафиксировать письменно
3. Занести итог в раздел «Источники проекта» в `SPEC.md`:
   - `Out-of-scope пункты (если есть): [список согласованных доп. пунктов]`
   - `Скоуп согласован: да — [дата]`
